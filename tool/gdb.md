# GDB的使用

## 常用命令

* file [文件名] :
    加载指定文件
* r :
 run 的简写，运行程序
* c :
    contiue的简写，继续执行程序
* b :设置断点
    用法：
  * b [行号]
  * b [函数名]
* d [删除断点]：删除断点
* p [变量名] ：显示指定变量的值
* where :显示当前运行的位置
* backtrace full ：显示栈帧和局部变量值
